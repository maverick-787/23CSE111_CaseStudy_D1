##  Problem Description

Managing school fees manually is time-consuming and error prone.
This system automates student records, fee payments, and tracking of dues, making the process efficient and organized.

---

##  How to Run the Code

1. Compile all Java files:

   ```
   javac school/fees/*.java
   ```
2. Run the program:

   ```
   java school.fees.Main
   ```
